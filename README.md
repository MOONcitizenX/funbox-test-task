# funbox-test-task

## Deployment:  
[Link to deployment](https://funbox-test-mooncitizenx.netlify.app/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/MOONcitizenX/funbox-test-task
```

Go to the project directory

```bash
  cd funbox-test-task
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
